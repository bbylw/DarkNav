# DarkNav

[English](README_EN.md) | 简体中文

优雅的黑暗主题导航网站，集成常用网站与工具。

## 特点

- 🌙 黑暗主题设计
- 🎨 现代化动画效果
- 📱 响应式布局
- 🌐 中英文支持
- ⚡ 纯静态部署

## 部署

### GitHub Pages

1. Fork 此仓库
2. 开启 GitHub Pages (Settings > Pages)
3. 选择 main 分支
4. 访问 `username.github.io/DarkNav`

### Cloudflare Pages

1. 创建新项目
2. 连接仓库
3. 部署设置：保持默认
4. 等待自动部署

## 开发

```bash
git clone https://github.com/your-username/DarkNav.git
cd DarkNav
python -m http.server 8000  # 或使用其他静态服务器
```
